TIC-TAC-TOE

A game that needs no introduction!!!Here you have an option to play with another palyer or a bot.

N-QUEENS

N-Queens is an implementation of an algorithm where we place the queens on an NxN board such that no two queens can attack each other!

PATH FINDING

In this section we can set source and destination on a board by drag and drop. On clicking on start button it starts searching for the path between source and destination.

PATTERN SEARCHING

In this section we can search for the pattern in the randomly generated board.We can search if there is any pattern with continuous matching characters with the input text in all four directions.(KMP Algorithm implementation)

PAC-MAN

In this section we will be able to play PAC-MAN game.This game further implements path finding algorithm which we used in previous sections inorder to help the ghosts reach the pac-man
